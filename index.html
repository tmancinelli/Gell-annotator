Welcome to the Gell annotator

This page will drive you to the generation of a XML/TEI document to map an area of the Gell manuscript.

Use the output of this tool to complete the annotation using an XML editor such as oXygen, VSStudio or VIm.

Your XML document, together with all the others from your collegues will map the entire Gell manuscript.

Refresh the page to start from scratch.

Choose the page you want to work on.

Choose the area you would like to work on.

Click on the image to add points. Then, unify them using the green button. Use the orange button to reset the selection.

Operation completed!
